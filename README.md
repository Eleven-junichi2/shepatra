# Shepatra

## Shepatra とは

Shepatra は、文字列をハッシュ化するアプリケーションです。

1つのハッシュ関数だけでなく、複数のハッシュ関数を順番に適用する手順を作り、その手順をレシピとして保存して再利用できます。  
同じ入力に対して、いつでも同じ手順でハッシュ値を生成できます。

## 何ができるの？

Shepatra では、主に次のことができます。

- 文字列をハッシュ化する
- 複数のハッシュ関数を順番に適用する
- その手順をレシピとして保存する
- 保存したレシピを選んで再利用する
- 生成したハッシュ値をクリップボードへコピーする

## どのように動くの？

Shepatra では、入力した文字列に対して、レシピに登録されたハッシュ関数を先頭から順番に適用します。

たとえば、あるレシピに 3 つのハッシュ関数が登録されている場合、最初のハッシュ結果を次の入力として使いながら、連続して変換していきます。  
最終的に得られた 16 進文字列が、出力結果になります。

## レシピとは？

レシピは、「どのハッシュ関数を、どの順番で使うか」を表す設定です。

Shepatra では、このレシピに名前を付けて保存できます。  
一度作成したレシピは、あとから選び直して同じ手順を再利用できます。

## 対応しているハッシュ関数

現在の実装では、次のハッシュ関数をレシピに含められます。

- SHA3_312
- BLAKE2b
- BLAKE3
- SHAKE_128
- SHAKE_256

## 使い方

### CLI 版

CLI 版では、レシピの作成と保存、保存済みレシピを使ったハッシュ生成ができます。

基本的な流れは次の通りです。

1. ハッシュ関数を選んで順番を決める
2. その組み合わせに名前を付けて保存する
3. 保存済みレシピを選ぶ
4. 文字列を入力してハッシュ値を生成する

生成したハッシュ値は表示され、クリップボードにもコピーされます。

### GUI 版

GUI 版では、保存済みレシピを選んで文字列をハッシュ化できます。

基本的な流れは次の通りです。

1. レシピを選ぶ
2. 文字列を入力する
3. ハッシュ値を確認する

生成したハッシュ値は画面に表示され、クリップボードにもコピーされます。

## 補足

現在の実装では、レシピの作成・保存は主に CLI 版で行います。  
GUI 版には設定タブがありますが、現時点ではレシピ編集用の機能は実装されていません。

## こんなときに使えます

- 同じ文字列に対して、決まった手順でハッシュ値を生成したいとき
- 複数のハッシュ関数を組み合わせて試したいとき
- ハッシュ化の手順を名前付きで保存して管理したいとき
- 同じ手順を繰り返し再利用したいとき
